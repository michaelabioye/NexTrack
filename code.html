The Code

How it works

This is my NexTrack Java. Code.

NexTrack is an event tracker that displays an array of statistics for various events. This UI/UX was built using 3 key languages; javaSCript, CSS, and HTML.

buildDropDown

Lets go over the 7 functions that I broke this code down in using javascript. The first function we have is buildDropDown. I used this function to display the cities that the events took place in by using a for loop to get the cities out of the array of objects. Then I used another for loop to display those cities once you click on the "All Events" button. I had a few duplicates of cities inside the array so I used a constructor function called new Set to help create an object with unique values that got rid of any duplicates. To display the drop down, I copied the element ID's to my javascript from my HTML page.

filterByCity

The second function I have is filterByCity this function was made to filter each city statistics by clicking on a specific city in the drop down option. I gave displayStats and displayEvents a parameter of filteredEvents to display the statistics of that particular city.

displayStats

In code number 3, displayStats has an array of objects in its parameter where I use a for loop to display the minimum , maximum, total, and average numbers of the attendees that were at said event.

displayEvents

In my fourth function displayEvents, I used a for loop to pull the objects out of the array. Once I had the objects out of the array, I cloned my tableRowEl and assigned a new variable and a querySelector to each class in the table template. After assigning the querySelectors and the new variables, I then went into my array of objects and pulled out event, city, state, date, and attendance to display my events in the table.

saveNewEvent

Number 5. saveNewEvent. I made this function so that any new events added to the form I created can be saved and displayed in both tables. So that the new data matches the data that is already stored in the array of objects, I converted the attendance string into a number, and I gave the function toLocaleDateString to my date. AFter that I push my new form to the getEvent fucntion and my saveEvents function as a backup in my getEvents if statement.

getEvents

As you may have seen throughout my code, the second to last function getEvents retrieve information in the form of strings from my saveEvents function. After retreiving the string of text, I turn the string of text back into an array of objects so that my page will still run smoothly. Note: I created an if statement for the soul purpose of having a backup plan set just incase I went to retrieve my string of text from the saveEvent function and it returned a value of null.

saveEvents

My last and final function on my javascript code is saveEvents. This is where my code will be stored and converted into a string of text. JSON.stringify converts my code to strings. Then I used setItem with a key value pair to finalize the code being stored.
